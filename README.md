## 🤝 介绍

对输入输出的消息进行预处理：唤醒增强、屏蔽用户、屏蔽报错、清洗文本、随机@ 、随机引用...

## 📦 安装

- 可以直接在astrbot的插件市场搜索astrbot_plugin_better_io，点击安装，耐心等待安装完成即可
- 若是安装失败，可以尝试直接克隆源码：

```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/Zhalslar/astrbot_plugin_better_io

# 控制台重启AstrBot
```
