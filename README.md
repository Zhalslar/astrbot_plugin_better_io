## 🤝 介绍

bot被禁言/解禁、被踢出群/拉群、被取消管理员/设为管理员时，会向主人打小报告

## 📦 安装

- 可以直接在astrbot的插件市场搜索astrbot_plugin_notice，点击安装，耐心等待安装完成即可
- 若是安装失败，可以尝试直接克隆源码：

```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/Zhalslar/astrbot_plugin_notice

# 控制台重启AstrBot
```

## ⌨️ 使用说明

请在配置中填写管理群的群号，以获得更有趣的体验！
